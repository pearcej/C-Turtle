Detailed Description

The AbstractTurtleScreen class is the abstract type for most turtle functionality. It intentionally excludes all input/output functionality, allowing for two intended derivates: an "interactive" screen, vs an "offline rendering" screen. Turtle class doesn't care which one it gets, in theory.
